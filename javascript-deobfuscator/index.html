  • Tujuan dari program ini adalah untuk membuat script yang dikaburkan lebih mudah dibaca dan membantu Anda memahami perilaku kasarnya, bukan untuk mencoba mereproduksi kode asli atau mengeluarkan kode yang dapat dieksekusi.
  • Program ini dibangun di atas nodejs dan babel.
  • Saya mengembangkan program ini untuk saya gunakan sendiri. Ini sering berakhir dengan kesalahan yang tidak tertangkap dan terkadang masuk ke loop tak terbatas dan membeku serta mengeksekusi beberapa kode script target di VM tanpa mempertimbangkan keamanan. VM diimplementasikan sebagai iframe[sandbox=""] dan dokumennya memiliki Content-Security-Policy:default-src 'self'; script-src 'unsafe-eval'. Anda dapat menguji keamanan/kerentanan VM dengan menempatkan kode apa pun di kotak input dan mengklik [Jalankan kode di vm(iframe)]. Lihat hasilnya di Alat Pengembang.
